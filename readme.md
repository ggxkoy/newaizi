# 桥梁防守视频玩法

## 当前需求

本项目当前目标是竖屏 3D 桥梁防守玩法，不再按原跑酷玩法开发。

- [当前玩法需求](docs/current-video-defense-requirements.md)
- [参考视频与元数据](docs/reference-video.md)
- [参考视频文件](media/reference/318-2%20FW_爆杀沙漠版.mp4)

当前核心规则：主角使用原项目小人资产，停在桥尾只做左右移动；桥面分成两条独立路线，左路仅为可攻击升级道具，右路仅有迎面敌人；主角自动直线向前射击，无自动索敌、无固定炮台。
